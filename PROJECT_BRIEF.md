# Working Version: Capsule Opening on Amoy Polygon

This version includes the following functionality:
- **NFT Capsule Minting:** Users can connect their MetaMask wallet to the Polygon Amoy testnet and mint a new 'Capsule' NFT.
- **Smart Contract Interaction:** The frontend is successfully connected to a deployed smart contract on the Amoy network.
- **Dynamic Inventory:** The user's inventory of Capsule NFTs is dynamically fetched and displayed on the website.
- **Metamask Integration:** The website properly handles wallet connection, disconnection, and network switching.
